Interactive Function Plotter

Explore how changing values affects a quadratic graph

f(x) = 1x² + 0x + 0

What's Happening?

Adjust the sliders to see how each coefficient affects the graph.

  • a (coefficient) - Controls the width and direction (upward/downward)
  • b (coefficient) - Controls the horizontal position of the vertex
  • c (constant) - Controls the y-intercept (where the graph crosses the y-axis)
Tip: Press Ctrl+S to save, Ctrl+L to load
