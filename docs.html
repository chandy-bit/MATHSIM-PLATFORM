MathSim Documentation

Welcome to the MathSim documentation. This guide will help you understand how to use the platform effectively, explore simulations, and maximize your learning experience.

Getting Started

MathSim is designed to be intuitive and easy to use. Follow these steps to begin your learning journey:

  1. Choose a subject - Browse through Algebra, Geometry, Calculus, Probability, or Pre-Calculus
  2. Select a simulation - Pick the topic you want to explore
  3. Adjust variables - Use sliders to change parameters and observe real-time changes
  4. Analyze results - Watch how the graph responds to your inputs
  5. Practice with games - Reinforce your learning through interactive challenges

Using Simulations

Our interactive simulations allow you to explore mathematical concepts visually. Here's how to use them:

Quadratic Function Simulator

The quadratic simulator lets you explore the function f(x) = ax² + bx + c:

  • a coefficient - Controls the width and direction (upward/downward) of the parabola
  • b coefficient - Controls the horizontal position of the vertex
  • c coefficient - Controls the y-intercept (where the graph crosses the y-axis)
Pro Tip: Try the preset buttons (U-Shape, ∩-Shape, Wide) to see common configurations.

Linear Function Simulator

The linear simulator explores y = mx + b:

  • m (slope) - Controls steepness and direction
  • b (intercept) - Controls where the line crosses the y-axis

Playing Games

Our Slope Challenge game helps you master slope identification through play:

  1. Look at the line displayed on the graph
  2. Choose the correct slope from the options
  3. Get points for correct answers
  4. Level up after consecutive correct answers
  5. Use hints if you're stuck

Tracking Progress

The progress dashboard gives you insights into your learning journey:

  • Tutorials Watched - Track your video learning
  • Games Played - See how often you practice
  • Accuracy - Monitor your performance over time
  • Best Score - Beat your personal records
  • Study Time - Track your dedicated learning time

Accessibility Features

MathSim is designed to be accessible to all learners:

  • High Contrast Mode - Improves visibility for users with visual impairments
  • Large Text Mode - Increases font size throughout the platform
  • Reduced Motion - Disables animations for users sensitive to motion
  • Dyslexic Font - Uses OpenDyslexic font for easier reading
  • Screen Reader Support - Full ARIA labels and semantic HTML
  • Keyboard Navigation - Complete functionality without a mouse

Frequently Asked Questions

Is MathSim free?

Yes! MathSim is completely free and open source. No subscription, no hidden fees.

Do I need to create an account?

No account required! Your progress is saved locally on your device. You can use MathSim immediately.

Does MathSim work offline?

Yes! MathSim is a Progressive Web App (PWA). Once you visit the site, it works offline. You can even install it as an app on your phone or computer.

What browsers are supported?

MathSim works on all modern browsers: Chrome, Firefox, Safari, and Edge. For the best experience, we recommend using the latest version of Chrome or Firefox.

How do I save my progress?

Your progress is automatically saved to your browser's local storage. You can also export your data from the Progress page for backup.

API Reference

MathSim provides a simple JavaScript API for advanced users:

// Change simulation parameters
                    MathSim.setSimulation('quadratic', { a: 2, b: -3, c: 1 });

                    // Track analytics event
                    analytics.trackEvent('custom', 'button_click', 'simulation');

                    // Get current user progress
                    const progress = analytics.getUserProgress();

                    // Export simulation graph as image
                    MathSim.exportGraph('png');
